## 如何轻松玩转这个仓库？

### 快速获取文件

1.  **打包下载:**
    *   "Download ZIP" (下载 ZIP) https://github.com/ut01/ITA-resources/archive/refs/heads/main.zip。
    *   下载完成后，解压一下，OK

2.  **技术流:**
    *   打开你的终端 (Terminal) 或命令提示符 (Command Prompt)。
    *   输入代码：
        ```bash
        git clone https://github.com/ut01/ITA-resources.git
        ```
    *   进度条显示 100% 就已经下载完成。
